# Contributing to Slipstream TypeScript SDK

First off, thanks for taking the time to contribute!

## Development Setup

1.  Use `npm install` to install dependencies.
2.  Run `npm test` to execute the test suite.
3.  Run `npm run build` to verify the build.
4.  Ensure code is formatted with `npm run format`.

## Pull Request Process

1.  Fork the repo and create your branch from `main`.
2.  Add tests for any new functionality.
3.  Ensure the test suite passes.
4.  Update the documentation.
5.  Sign your commits (DCO).

## Code Style

We follow standard ESLint and Prettier configurations included in the project.
